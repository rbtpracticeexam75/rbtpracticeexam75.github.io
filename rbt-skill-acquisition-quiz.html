1. Which of the following is the best example of a skill acquisition program?
Skill acquisition programs focus on teaching new skills, such as self-care, communication, or academic skills, rather than assessing or tracking behavior.
2. When designing a skill acquisition program, the first step is to:
Baseline data helps determine the client’s current skill level before teaching begins, which is essential for measuring progress.
3. A discrete trial teaching (DTT) session is best described as:
DTT involves breaking skills into small steps and practicing them repeatedly in a structured way to teach new skills efficiently.
4. Prompt fading is used to:
Prompt fading gradually decreases assistance so the client can perform the skill independently.
5. Which reinforcement schedule is typically used during initial skill acquisition?
Continuous reinforcement (rewarding every correct response) helps learners acquire new skills faster before moving to intermittent schedules.
