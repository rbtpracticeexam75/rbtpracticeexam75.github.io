RBT Documentation Quiz

Click an answer to see if you're correct and view the explanation!

1. What is the primary purpose of data collection in ABA?
Data collection allows BCBA and RBTs to make informed decisions about the client’s treatment and track progress objectively.
2. How often should RBTs record session data?
Recording data immediately ensures accuracy and prevents loss of important information.
3. Which of the following is NOT appropriate documentation practice?
Falsifying data is unethical and violates professional standards.
4. Which format is commonly used to record behavioral data?
Structured data sheets and graphs provide objective, measurable, and trackable information.
5. What should an RBT do if data collection is unclear?
RBTs should clarify with the supervising BCBA to ensure accurate documentation.
